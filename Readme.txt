
This project is about how to develop a report using database where user can enter order number and generate pdf file. Using Python/DJango/SQLLight/ReportLab and other appropriate libraries.

run "py manage.py runserver" from windows explorer.

open browser with http://127.0.0.1:8000/

enter 1 as order number and download.
